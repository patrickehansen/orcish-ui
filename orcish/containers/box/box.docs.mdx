import { Box } from './box';

## React Component for rendering a box

A basic div that renders some text

### Component usage
```js
<Box >
    <p>Hello from Basic Box</p>
</Box>
```

### Using props to customize the box

Modify the text to see it change live:
```js live
<Box p={'20px'}>
    <p>Hello from Basic Box</p>
</Box>
```
